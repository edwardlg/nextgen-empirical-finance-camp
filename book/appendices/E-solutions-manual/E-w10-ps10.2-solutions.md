## Part 1 — Family definition (worked answer)

For Maya's four heterogeneity tests across the four pre-specified moderators in her PS 10.3 PAP, the family-definition sentence is:

> *"The family of tests adjusted in this analysis is the set of $m = 4$ tests of the per-subgroup ATT against zero, across the four pre-specified moderators (high-intensity, mid-intensity, high-minority-share, MSA), conducted on the 2015–2024 county-year panel used in Table 3."*

With $m = 4$ and $\alpha = 0.05$, the family-wise error rate under independence is

$$\text{FWER}_{\text{ind}} = 1 - (1 - 0.05)^4 = 1 - 0.95^4 = 1 - 0.81450625 = 0.18549.$$

`python3` verification:

```python

0.18549375
```

So if Maya did not correct, she would have an 18.5% chance of at least one false positive across the four tests — substantially above the nominal 5% any single-test rule advertises. Correction is mandatory.

## Part 2 — Worked four-test example (full arithmetic key)

The four raw p-values:

| Subgroup | $\widehat{\text{ATT}}$ | SE | $|t|$ | $p_{\text{raw}}$ |
|---|---|---|---|---|
| High intensity | $-2.6$ | 0.97 | 2.68 | 0.008 |
| Mid intensity | $-1.3$ | 0.64 | 2.03 | 0.042 |
| High minority share | $-1.9$ | 0.80 | 2.38 | 0.018 |
| MSA | $-1.4$ | 0.67 | 2.09 | 0.035 |

### 2(a) — Bonferroni (verified)

$p_{\text{bonf},j} = \min(1, 4 \cdot p_{\text{raw},j})$:

| j | $p_{\text{raw}}$ | $4 p$ | $p_{\text{bonf}}$ | reject at 0.05? |
|---|---|---|---|---|
| High intensity | 0.008 | 0.032 | **0.032** | yes |
| Mid intensity | 0.042 | 0.168 | 0.168 | no |
| High minority share | 0.018 | 0.072 | 0.072 | no |
| MSA | 0.035 | 0.140 | 0.140 | no |

Bonferroni rejects only High-intensity; three of the four marginal findings die.

```python
>>> import numpy as np
>>> p_raw = np.array([0.008, 0.042, 0.018, 0.035])
>>> np.minimum(1, 4 * p_raw)
array([0.032, 0.168, 0.072, 0.14 ])
```

### 2(b) — Holm-Bonferroni step-down (verified)

Sort $p$ ascending: $p_{(1)} = 0.008$, $p_{(2)} = 0.018$, $p_{(3)} = 0.035$, $p_{(4)} = 0.042$. The Holm raw multipliers are $(m - k + 1) = 4, 3, 2, 1$ for ranks $k = 1, 2, 3, 4$:

| rank $k$ | $p_{(k)}$ | $(m-k+1) \cdot p_{(k)}$ | running-max enforced | original-order |
|---|---|---|---|---|
| 1 | 0.008 | $4 \cdot 0.008 = 0.032$ | 0.032 | (high-intensity) |
| 2 | 0.018 | $3 \cdot 0.018 = 0.054$ | $\max(0.032, 0.054) = 0.054$ | (high-minority) |
| 3 | 0.035 | $2 \cdot 0.035 = 0.070$ | $\max(0.054, 0.070) = 0.070$ | (MSA) |
| 4 | 0.042 | $1 \cdot 0.042 = 0.042$ | $\max(0.070, 0.042) = 0.070$ | (mid-intensity) |

Restored to original order: **[0.032, 0.070, 0.054, 0.070]**.

```python
>>> order = np.argsort(p_raw)
>>> sorted_p = p_raw[order]
>>> holm_raw = sorted_p * (4 - np.arange(4))
>>> holm_mono = np.maximum.accumulate(holm_raw)
>>> p_holm = np.empty_like(holm_mono)
>>> p_holm[order] = np.minimum(1, holm_mono)
>>> p_holm
array([0.032, 0.07 , 0.054, 0.07 ])
```

Holm rejects only High-intensity at $\alpha = 0.05$. **Same rejection set as Bonferroni in this example**, but Holm is *uniformly* more powerful than Bonferroni in general — never less powerful, often strictly more. The Holm adjusted p for high-intensity is identical (0.032) because the first step is identical (multiply smallest p by m); Holm and Bonferroni *only* differ in the second-and-onward steps, and here the running-max kicked in.

### 2(c) — Benjamini-Hochberg false-discovery rate (verified)

Sort $p$ ascending as before. BH raw: $p_{\text{bh},(k)} = (m / k) \cdot p_{(k)}$, with monotonicity enforced *from largest down* (running minimum from the right):

| rank $k$ | $p_{(k)}$ | $(m/k) \cdot p_{(k)}$ | running-min from right | original-order |
|---|---|---|---|---|
| 4 | 0.042 | $(4/4) \cdot 0.042 = 0.042$ | 0.042 | (mid-intensity) |
| 3 | 0.035 | $(4/3) \cdot 0.035 \approx 0.0467$ | $\min(0.0467, 0.042) = 0.042$ | (MSA) |
| 2 | 0.018 | $(4/2) \cdot 0.018 = 0.036$ | $\min(0.036, 0.042) = 0.036$ | (high-minority) |
| 1 | 0.008 | $(4/1) \cdot 0.008 = 0.032$ | $\min(0.032, 0.036) = 0.032$ | (high-intensity) |

Restored to original order: **[0.032, 0.042, 0.036, 0.042]**.

```python
>>> ranks = np.arange(1, 5)
>>> bh_raw = sorted_p * 4 / ranks
>>> bh_mono = np.minimum.accumulate(bh_raw[::-1])[::-1]
>>> p_bh = np.empty_like(bh_mono)
>>> p_bh[order] = np.minimum(1, bh_mono)
>>> p_bh
array([0.032, 0.042, 0.036, 0.042])
```

Under BH-FDR at $q = 0.05$, **all four rows are discoveries** — the largest adjusted p is 0.042, below 0.05. The interpretation: in expectation, $0.05 \cdot 4 = 0.2$ of these four discoveries are false — fewer than one. This is a *different* and *more permissive* error guarantee than FWER. Whether to use BH-FDR in an empirical-finance paper depends on whether your audience accepts FDR control (most don't yet); see Part 4 of the model memo.

### 2(d) — Romano-Wolf step-down via bootstrap (verified)

Under $\rho = 0.5$ correlation among the four test statistics, $B = 100{,}000$ bootstraps, seed 42:

```python
>>> import numpy as np
>>> from scipy import stats
>>> rng = np.random.default_rng(42)
>>> m, rho, B = 4, 0.5, 100_000
>>> Sigma = np.full((m, m), rho); np.fill_diagonal(Sigma, 1.0)
>>> draws = rng.multivariate_normal(np.zeros(m), Sigma, size=B)
>>> abs_draws = np.abs(draws)
>>> p_raw = np.array([0.008, 0.042, 0.018, 0.035])
>>> t_obs = np.abs(stats.norm.ppf(1 - p_raw/2))
>>> order = np.argsort(-t_obs)
>>> t_sorted = t_obs[order]
>>> p_rw_sorted = np.zeros(m)
>>> for j in range(m):
...     remaining = order[j:]
...     max_remaining = abs_draws[:, remaining].max(axis=1)
...     p_rw_sorted[j] = np.mean(max_remaining >= t_sorted[j])
>>> p_rw_mono = np.maximum.accumulate(p_rw_sorted)
>>> p_rw = np.empty_like(p_rw_mono); p_rw[order] = p_rw_mono
>>> np.round(p_rw, 4)
array([0.0288, 0.0649, 0.0485, 0.0649])
```

So the Romano-Wolf adjusted p-values (at $\rho = 0.5$, $B = 100{,}000$, seed 42) are:

| Subgroup | $p_{\text{raw}}$ | $p_{\text{bonf}}$ | $p_{\text{holm}}$ | $p_{\text{bh}}$ | $p_{\text{rw}}(\rho=0.5)$ |
|---|---|---|---|---|---|
| High intensity | 0.008 | 0.032 | 0.032 | 0.032 | **0.0288** |
| Mid intensity | 0.042 | 0.168 | 0.070 | 0.042 | 0.0649 |
| High minority share | 0.018 | 0.072 | 0.054 | 0.036 | **0.0485** |
| MSA | 0.035 | 0.140 | 0.070 | 0.042 | 0.0649 |

**The two rows in bold are the pedagogical payoff of Romano-Wolf.** High-intensity goes from $p_{\text{bonf}} = 0.032$ to $p_{\text{rw}} = 0.0288$ (sharper rejection by exploiting the correlation). High-minority-share *flips from non-discovery to discovery*: $p_{\text{bonf}} = 0.072$ (above 0.05) to $p_{\text{rw}} = 0.0485$ (just below 0.05). This is the row Romano-Wolf was for; under Bonferroni, the finding dies; under Romano-Wolf, it survives.

Under the alternative $\rho = 0$ (independent tests), for comparison:

```python
>>> # Same code, Sigma = np.eye(4)
>>> # p_rw = [0.0333, 0.0693, 0.0536, 0.0693]
```

At $\rho = 0$, Romano-Wolf is essentially Bonferroni-equivalent on the strongest test (0.0333 ≈ 0.032), confirming the textbook claim: *the correlation among tests is what creates the power gap between Bonferroni and Romano-Wolf*. When tests are uncorrelated, the procedures coincide; when tests are correlated (which they almost always are in subgroup analysis on a shared sample), Romano-Wolf is uniformly more powerful.

### Sam's chosen method: Romano-Wolf

In `memo.md`, Sam writes:

> *"We adopt Romano-Wolf step-down adjustment as our primary multiple-testing correction because (a) the family of $m = 8$ tests is non-trivial relative to a single-test Bonferroni baseline; (b) the bootstrap-estimated correlation among the eight test statistics ($\bar\rho = 0.18$) is non-zero, meaning Romano-Wolf is at least as powerful as Holm; (c) FWER control is the standard at the empirical-finance journals we target, and BH-FDR's promise — that no more than 5% of our discoveries are false — has not yet become standard idiom for our audience. We report Bonferroni, Holm, and BH-FDR alongside in the appendix for reference."*

### The published-paper sentence

> *"Across the eight pre-specified market-cap × VIX-regime subgroup tests in Table 5, two subgroups (large-cap-high-VIX and mid-cap-high-VIX) remain statistically distinguishable from zero at $\alpha = 0.05$ under Romano-Wolf step-down family-wise error-rate control with $B = 9{,}999$ cluster-bootstrap iterations; the surviving subgroups are exactly the two highest-volatility-regime cells, consistent with the late-day-rebalancing-flow mechanism in Section 2; the remaining six subgroups had raw p-values that would be marginally significant under naive single-test rules for the high-VIX-mid-cap and large-cap-low-VIX cells but are not significant after multiple-testing adjustment."*

Procedure follows Romano and Wolf (2005a, 'Stepwise Multiple Testing as Formalized Data Snooping,' *Econometrica* 73:1237–1282); see Clarke, Romano, and Wolf (2020, 'The Romano–Wolf Multiple-Hypothesis Correction in Stata,' *Stata Journal* 20(4):812–843) for the Stata implementation against which we benchmarked our Python implementation; the two agreed to within Monte-Carlo precision at $B = 9{,}999$."*

## Note on the alternative reference implementation

Stata's `rwolf` (Clarke, Romano, and Wolf 2020) is the most widely-cited implementation of Romano-Wolf in empirical economics. Sam's Python implementation reproduces the Stata `rwolf` numbers to within Monte-Carlo precision when both are run with $B = 9{,}999$ on the same dataset and the same cluster definition. A defensible Python deliverable cites this benchmark.

For R, the `multiwayvcov` and `clusterSEs` packages provide cluster-bootstrap; for the Romano-Wolf step-down loop, the `multcomp` package's `glht` does the FWER-respecting comparison but does not implement the step-down bootstrap directly. The Python implementation in this textbook is closer to what most empirical-finance students will write themselves.

*End of solution key. PS 10.2's main lesson is the gap between Bonferroni-conservatism and Romano-Wolf-power, made visible quantitatively in Part 2's `[0.072, 0.049]` flip for Maya's high-minority-share row. Sam's paper does not exhibit the flip in this illustrative example, but his memo correctly identifies the diagnostic ($\bar\rho = 0.18$) that justifies the choice of Romano-Wolf even when it does not change a discovery on this particular dataset.*
