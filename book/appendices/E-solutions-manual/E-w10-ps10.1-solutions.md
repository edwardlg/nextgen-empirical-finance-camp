### 3. The arithmetic worked end-to-end (verified with `python3`)

```python
# triage_arithmetic.py — verified Aug 17 against Maya's 45-row ledger
import numpy as np
# Cost buckets: L=3h, M=10h, H=30h (hours_estimate midpoints)
cost_h = {"L": 3, "M": 10, "H": 30}
# Verdict counts (from the 27-cell distribution above):
# Required = 4 central + Required-high-value-low-cost rows
# By inspection of the 45-row ledger, Required = 7 items (4 central + 3 H-value-L-cost)
# Welcome = 14 items
# Decline = 24 items
# Hours per bucket (Required is 4 L-C + 1 M-C + 2 M-H-N + others):
H_required = 3 + 3 + 3 + 10 + 10 + 12 + 10   # the 7 Required rows
H_welcome  = 4 + 1 + 3 + 4 + 1 + 4 + 3 + 3 + 4 + 3 + 1 + 1 + 4 + 3  # 14 Welcome
H_decline  = 30 + 30 + 30 + 30 + 30 + 10 + 10 + 10 + 10 + 10 + 3 + 3 + 3 + 3 + 3 + 3 + 3 + 3 + 3 + 3 + 0 + 0 + 0 + 0
H_all = H_required + H_welcome + H_decline
print(f"H_required = {H_required}")  # 51
print(f"H_welcome  = {H_welcome}")   # 39
print(f"H_decline  = {H_decline}")   # 230
print(f"H_all      = {H_all}")        # 320
print(f"H_req/80   = {H_required/80:.2f}")  # 0.638  (Required fits)
print(f"Slack S    = {80 - H_required}")     # 29
print(f"H_all/80   = {H_all/80:.2f}")        # 4.0 — the compression factor
print(f"Decline %  = {100*H_decline/H_all:.1f}%")  # 71.9% (above the 30-60% band; see note)
```

**Interpretation.** The total if-you-did-everything is 320 hours — four times Maya's 80-hour budget; the triage compression factor is 4×. The Required items fit comfortably (51 hours / 80 = 64% utilization), leaving 29 hours of slack. The Welcome pool needs 39 hours but only 29 are available, so Maya will execute Welcome items in `vph` order (Part 2(d) below) until slack is exhausted.

The 71.9% decline percentage is *above* the recommended 30–60% band, which is normal for a dataset where most comments are H-cost-L-value or L-cost-L-N (citation chases that aren't on the paper's threats table). Maya documents this in the memo: *"The high decline rate reflects an unusually large pool of presentation-comments (font, framing, citation chases) that do not bear on the threats table; the four central-threat items are all Required."* That sentence is the defense.
